== Description ==

Theme Canary Demo Import is a simple demo importer WordPress plugin that allows you to import the dummy content of Canary Themes within a few click

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Theme Canary Demo Importer' and install the plugin
3. You need to add One Click Demo Import' plugin so that Theme Canary Demo Import will work
3. Activate both of the plugin from your Plugins page.

== Frequently Asked Questions ==

= I have activated the plugin. Where is the "Import Demo Data" page? =

You will find the import page in dashboard > Appearance > Import Demo Data

= What does this plugin do? =

This plugin will help you to import demo content for themecanary.com

= Will I need any other plugin? =

Yes, To work with this plugin, you need to activated One click demo import plugin.
